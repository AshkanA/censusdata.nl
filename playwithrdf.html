LEARN SPARQL BY COPY PASTING USING VARIABLES MAPPINGS

 

We use this SPARQL query as the template to query the census variables across the years. By illustirating the link between the variables (mappings) we show how we are able to modify this query for various variables across the years. We aim to illustrate and stimiulate use of SPARQL queries by providing the possible variable combinations.

 

By using a template query and an overview of all the different variable combinitions, we show that we only have to change a couple of lines to get valid query results.


1. look at mappings

Bewoonde Huizen, Onbewoonde Huizen en Huizen in aanbouw do not have a connection to other variables in our harmonization. So, in order to query for the number of inhabited houses we have to query for the variable group name: HousingType and value: Bewoondehuizen.

query examples

In contrast to the previous examples the other variables have specific relations which need to be adhered. For example we see that HousingTypes such as Ships and Caravans are related to the ResidenceStatus variable group. This also applies for the demographic variable Sex.
